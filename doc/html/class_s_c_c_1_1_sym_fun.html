Detailed Description

Instances of SCC::SymFun provide means of evaluating functions of double variables that are specified symbolically, i.e. as string. The names of the variables and symbolic constants used to express the function symbolically are specified by the programmer. The operations allowed in the function specification include the standard algebraic operations as well as many of the standard functions available for double values in the C++ standard library.

The operators allowed are

{"+", "-", "+", "-", "*", "/", "^", "sin", "cos", "tan","asin","acos","atan","atan2", "sinh","cosh","tanh", "ceil","exp","abs","floor","fmod","log","log10","sqrt","pow"};

The "^" is interpreted as the exponentiation operator, i.e. x^2 is x squared.

Required version of C++ : >= C++11

Creates a SymFun instance from the initialization string S. Variables names are specified by the elements of V and symbolic constant names are specified in C. The initial values of the symbolic constants are specified in the vector of double values Cvalues.

If the construction process fails, program execution stops and an error message is output. See initialize(...) member functions for creation with exception handling.

  • V : vector<string> specifying independent variable names
  • S : std::string specifying the function
  • C : vector<string> specifying symbolic constant names
  • Cvalues : vector<double> specifying symbolic constants values

Sample demonstrating creation and usage of an instance that implements a*x^2 + b*x + c.

//
// Create a SCC::SymFun that implements a*x^2 + b*x + c;
// a, b, c being symbolic constants.
//
vector<string> V = {"x"}; // specify variable name
vector<string> C = {"a","b","c"}; // specify constant names
vector<double> Cvalues = {1.0, 2.0, 1.0}; // initial values of a,b,c
string S = "a*x^2 + b*x + c"; // initialization string
SCC::SymFun F(V,C,Cvalues, S);
cout << F << endl << endl; // print out function
cout << "The value of the function at x = 1.0 is "
<< F(1.0) << endl << endl;
F.setConstantValue("a",1000.0); // reset the symbolic constants a and c
F.setConstantValue("c",2000.0);
cout << F << endl << endl; // print out function
cout << "The value of the function at x = 1.0 is "
<< F(1.0) << endl;
